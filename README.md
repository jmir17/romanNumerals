# Roman Numerals Kata

Write a program that takes a number as argument and it returns the number in roman number.

Example results:
```
1 >> I
2 >> II
3 >> III
4 >> IV
5 >> V
9 >> IX
21 >> XXI
50 >> L
97 >> XCVII
100 >> C
500 >> D
1000>> M
```

To run the tests, execute:  
`npm i` to install the depencencies and then  
`npm run test` to run the tests
